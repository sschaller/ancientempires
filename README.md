# Ancient Empires Clone Made With Phaser

## Play Now
You can find a demo here: [https://sschaller.github.io/ancientempires/](https://sschaller.github.io/ancientempires/).

## Deployment
1. Install project with `npm install`
2. Compile and minify project using `npm run all`
3. Copy assets, dist/app.min.js and index.html and run on webserver

```
    .
    ├── assets                  # Game asset files
    ├── dist                    # Compiled files
    ├── src                     # Source files
    └── README.md
```
